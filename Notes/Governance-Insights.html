dashboard LaunchDarkly Governance Dashboard

infoOverview

This dashboard provides actionable insights into LaunchDarkly feature flags, helping teams manage flag lifecycle, technical debt, and governance. It visualizes flag data using interactive charts and tables, making it easy to identify priorities, risks, and cleanup opportunities.

calculateData Flow & Calculation

  1. Data Fetching: cloud_download The dashboard fetches all flags using the LaunchDarkly API, then processes raw flag data via analyzeFlags().
  2. Flag Analysis: analytics
    • Age (days): calendar_today Days since flag creation
    • Lifecycle Stage: timeline Determined by age, archived status, and custom rules
    • Priority Score: star_rate Composite score based on age, lifecycle, and other attributes
    • Temporary/Permanent: hourglass_empty Classified based on flag property
    Aggregated Metrics: bar_chart
    • Total flags, temporary/permanent counts, archived flags
    • Age buckets: 0-30, 31-90, 91-180, 180+ days
    • Flags ready to archive, high-priority, cleanup candidates

insert_chartChart Components & Logic

  1. Priority Line Chart: Shows priority score vs age (X: Age, Y: Priority, color/size: urgency). Tooltips list maintainers & flag names.
  2. Lifecycle Chart: Bar or pie visualization of flags by lifecycle stage (Live, Review, Archived).
  3. Age Distribution Chart: bar_chart Bar chart of age buckets to spot stale flags and technical debt.
  4. Flag Types Chart: Pie/bar chart by flag type (boolean, string, number, json).
  5. Governance Insights Cards: insights Cards with ratios of temporary flags, old flags (>180 days), and average priority score, styled by risk level.
